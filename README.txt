# SW1E Foundry System

Unofficial Foundry VTT support for **Star Wars: The Roleplaying Game, 1st Edition**.

## Overview

This project is a Foundry implementation of the West End Games 1E rules framework for Star Wars roleplaying.

The implementation focus is on the rules areas most useful for actual system work:

- die codes and pips
- opposed rolls and difficulty numbers
- action modifiers, multiple actions, and reaction skills
- personal combat
- wounds and healing
- starships and astrogation
- Force points, Dark Side points, Force skills, and powers
- stock NPC support
- character templates
- weapon and reference charts

## Status

Work in progress.

This system is being built to support play at the table while preserving the structure of the original WEG 1E rules.
